### 4. 右侧溯源区的“逆向拾取” (Reverse Extraction) - 可选的超极简彩蛋

右侧的 40% 区域目前是“只读面板”。但在清洗语料时，用户经常会看着右侧的原文突然发现：“哎，这个词模型居然漏抓了，我也想加进生词本”。

- **轻量优化**：在右侧面板允许用户**划词选中 (Highlight)** 某一段文本。
    
- 选中文本后，鼠标指针附近立即弹出一个极简的悬浮动作：`[ + Add Expression ]`。
    
- 点击后，左侧列表顶部（或当前聚类组下方）立刻插入一张新卡片，自动将刚选中的词填入 `Source`，并等待用户输入 `Target`。这种“指哪打哪”的体验会让整个工具的掌控感直线飙升。
